Multiplication table

Dec 21, 2017 in R-Programming

R로 구구단을 출력해보는 연습을 해보겠습니다.

이를 위해선 반복문/조건문의 문법을 숙지하고 있어야 합니다.

반복문

동일한 연산을 여러번 반복하기 위해 사용합니다.
여기서는 for을 사용하겠습니다.

조건문

구구단 계산 시 홀수/짝수만 출력하고 싶을 때 사용합니다.
‘if(조건식) 연산’의 형태로 사용하며, 조건문과 결합하여,
TRUE일 경우 해당연산을 실행하도록 만들 수 있습니다.

1. 구구단 계산하기 (일반)

2. 구구단 계산하기 (짝수)
